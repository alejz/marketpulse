# Project Brief: AI Marketing Agent

## Purpose

Otonom dijital pazarlama ve sosyal medya yönetim agenti. Kullanıcının sosyal medya hesaplarını otomatik olarak yöneten, içerik üreten ve reklam veren bir AI sistemi.

## Target Users

- Dijital pazarlamacılar
- Sosyal medya yöneticileri
- E-ticaret işletmeleri
- Teknoloji ürünü satan işletmeler

## Core Features

### 1. Sosyal Medya Hesap Yönetimi
- Çoklu platform desteği (Instagram, Twitter, LinkedIn, Facebook, TikTok)
- Hesap bağlama ve takip
- Otomatik paylaşım açma/kapama
- Durum izleme (aktif/paused/error)

### 2. İçerik Üretimi
- AI destekli içerik oluşturma (GPT-4/Claude/Gemini)
- Farklı içerik tipleri (gönderi, hikaye, reel, thread, caption)
- Ton seçimi (profesyonel, eğlenceli, samimi, ciddi, teknik)
- Otomatik hashtag ekleme
- Prompt şablonları

### 3. Medya Yönetimi
- Fotoğraf/video yükleme
- AI görsel üretimi (DALL-E, Midjourney, Stable Diffusion)
- Stil seçimi
- Medya kütüphanesi

### 4. Kampanya Yönetimi
- Kampanya oluşturma
- Bütçe planlaması
- Hedef kitle seçimi
- Performans takibi
- Görsel şablonları

### 5. Reklam Yönetimi
- Çoklu platform reklamları (Google, Meta, LinkedIn, TikTok)
- Performans metrikleri
- Otomatik optimizasyon
- A/B test automation

### 6. Agent Otomasyonu
- Otonom çalışma modu
- Otomatik yanıtlama
- Trend takibi
- Zamanlama optimizasyonu
- Raporlama

## Key Requirements

### Must Have
- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4
- Modern dashboard UI
- Çoklu platform entegrasyonu hazırlığı
- AI içerik üretimi arayüzü
- Agent durumu takibi

### Nice to Have
- Gerçek API entegrasyonları
- Veritabanı persistence
- Gelişmiş analytics
- Raporlama sistemi

## Success Metrics

- Clean, zero-error TypeScript setup
- Passing lint and type checks
- Kullanıcı dostu dashboard arayüzü
- Tüm fonksiyonların çalışır durumda olması

## Constraints

- Frontend odaklı (API entegrasyonları sonraki aşamada)
- Next.js 16 + React 19 + Tailwind CSS 4
- Package manager: Bun
